## Usage

```sh
+--------------+
| Command help |
+--------------+
-v  : This Program Version
-u <UA-code> : User-Agent prefix
<UA-code>
a6   : android 6.x
a7   : android 7.x
s    : Apple Safari
c    : Google Chrome
i    : Internet Explorer
f    : Mozzila FireFox
[User-Agent Usage]
$ python requndres.py -u s http://example.com/
$ python requndres.py -u i http://example.com/
```
